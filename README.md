**개발 기간**: 2024.03.14 ~ 2024.03.28 (2주)  
**팀 구성**: 5명  
**담당 업무**: 스테이지 1 레벨 디자인 및 게임플레이 시스템 구현  

## 기술 스택
- Unity3D, C#, Visual Studio 2022, Figma

## 주요 구현 사항
- 단계별 난이도 증가 스폰 시스템 (Level1 → Level2 → Boss)
- 동적/정적 장애물을 활용한 전략적 맵 디자인
- 3D 공간 오디오 시스템
- 레벨별 차등화된 적 AI 패턴
- HashSet을 활용한 오브젝트 상태 관리
